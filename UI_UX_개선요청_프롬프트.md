# Rachgia Factory 통합 생산관리 대시보드 UI/UX 개선 요청

## 프롬프트 (아래 내용을 Claude에게 전달하세요)

---

첨부한 HTML 파일은 베트남 Rachgia 공장의 신발 생산 현황을 통합 모니터링하는 대시보드입니다.

### 대시보드 개요
- **목적**: Factory A, B, C, D 4개 공장의 Loadplan 데이터를 시각화하고 생산 현황 모니터링
- **데이터**: 4,124건의 오더 데이터 (2025년 11월 ~ 2026년 4월)
- **사용자**: 생산관리팀, 물류팀, 경영진
- **기술스택**: HTML + Tailwind CSS + Chart.js + SheetJS (단일 파일, CDN 사용)

### 현재 구현된 기능

#### 1. 상단 요약 카드
- 총 오더 건수, 총 주문 수량, 최종 출고 완료, 전체 완료율 (도넛 차트)

#### 2. 공정별 생산 현황 (8개 공정 Progress Bar)
- 재단 → 재봉전 → 재봉투입 → 재봉 → 샘플핏 → 조립 → 창고입고 → 창고출고
- 각 공정별 완료율과 완료 수량 표시

#### 3. 공장별 출고 완료율
- Factory A, B, C, D 각각의 출고 완료율 카드
- 완료율에 따른 색상 구분 (초록/노랑/빨강)

#### 4. 중요 행선지 현황
- 일본, 한국, 중국, 대만, 인도 5개국 고정
- 국가별 수량, 건수, 완료율 Progress Bar
- 한국명 표시 (Japan→일본, South Korea→한국, China→중국, Taiwan→대만, India→인도)

#### 5. 출고 예정일별 현황
- 날짜별 예정 수량 표시
- 지연 오더 하이라이트 (빨간색)

#### 6. 필터 기능
- 월별, 행선지별, 공장별, 완료 상태별, 텍스트 검색

#### 7. 탭 구성
- 월별 현황: 막대 차트 + 테이블
- 행선지별 현황: 도넛 차트 + 상세 테이블
- 모델별 현황: 가로 막대 차트 + 테이블
- 공장별 현황: 비교 차트 + 공정별 상세
- 히트맵: 모델×행선지 수량 히트맵, 중요 행선지×SDD월 히트맵, 중요 행선지×모델 히트맵
- 상세 데이터: 공정별 상태 아이콘 표시, Excel 다운로드

#### 8. 모달
- 오더 상세 보기 (공정별 타임라인)
- 목록 모달 (공장/국가/모델별)

### UI/UX 개선 요청사항

다음 관점에서 개선이 필요한 부분을 분석하고, 구체적인 개선안을 제안해주세요:

#### 1. 시각적 디자인 (Visual Design)
- 색상 체계 및 일관성
- 타이포그래피 및 가독성
- 여백 및 레이아웃 밸런스
- 아이콘 및 시각적 요소 활용
- 모던하고 프로페셔널한 인상
- Progress Bar 디자인 개선

#### 2. 사용성 (Usability)
- 필터 UX 개선점
- 데이터 탐색 편의성
- 클릭/터치 영역 및 피드백
- 정보 계층 구조
- 핵심 데이터로의 접근성
- 공정별 상태 표시 개선

#### 3. 데이터 시각화 (Data Visualization)
- 차트 유형 적절성
- 색상 팔레트 개선
- 레이블 및 범례 가독성
- 인터랙티브 요소 강화
- 데이터 스토리텔링
- 공정 흐름 Funnel Chart 추가 제안

#### 4. 반응형 디자인 (Responsive Design)
- 모바일 환경 최적화
- 태블릿 레이아웃
- 차트 크기 조정
- 터치 인터랙션

#### 5. 성능 및 접근성 (Performance & Accessibility)
- 대용량 데이터 렌더링 최적화 (4,000+ 레코드)
- 접근성(a11y) 개선점
- 로딩 상태 및 피드백

#### 6. 추가 기능 제안
- 사용자 경험을 높일 수 있는 새로운 기능
- 생산 관리자에게 유용한 인사이트 제공 방법
- 업무 효율성 향상 아이디어
- 알림/경고 시스템 제안
