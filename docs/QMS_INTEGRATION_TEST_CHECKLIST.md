# QMS (Quality Management System) Integration Test Checklist

**Version**: v18
**Date**: 2026-01-18
**Tester**: _______________

## Pre-requisites
- [ ] Google Cloud Console에 프로젝트 설정됨
- [ ] OAuth 2.0 Client ID 생성됨
- [ ] Google Sheets API 활성화됨
- [ ] 테스트용 Google Sheets 문서 생성됨 (3개 시트: QMS_Targets, QMS_Tasks, QMS_Results)

---

## Phase 1: 기본 UI 테스트

### 1.1 QMS 탭 로딩
- [ ] 대시보드 로드 후 "🎯 품질관리" 탭 표시됨
- [ ] 탭 클릭 시 QMS 콘텐츠 영역 표시됨
- [ ] 5개 서브탭 모두 표시됨:
  - [ ] 📋 타겟등록
  - [ ] 📊 모니터링현황
  - [ ] 📝 작업지시
  - [ ] 📑 작업목록
  - [ ] ✅ 결과관리

### 1.2 서브탭 전환
- [ ] 각 서브탭 클릭 시 해당 콘텐츠 표시됨
- [ ] 이전 서브탭 콘텐츠 숨겨짐
- [ ] 활성 탭에 하이라이트 스타일 적용됨
- [ ] 다크모드에서 정상 표시됨

### 1.3 툴바 UI
- [ ] 🔄 동기화 버튼 표시됨
- [ ] ⚙️ 설정 버튼 표시됨
- [ ] 로그인 전: "로그인 필요" 표시됨
- [ ] 동기화 버튼 클릭 시 로그인 유도 또는 동기화 실행

---

## Phase 2: Google 인증 테스트

### 2.1 설정 모달
- [ ] ⚙️ 버튼 클릭 시 설정 모달 열림
- [ ] 모달에 다음 필드 표시됨:
  - [ ] Spreadsheet ID 입력 필드
  - [ ] Client ID 입력 필드
  - [ ] 자동 동기화 토글
  - [ ] 동기화 간격 선택 (토글 ON 시)
  - [ ] 마지막 동기화 시간 표시
- [ ] ESC 키로 모달 닫힘
- [ ] 외부 클릭으로 모달 닫힘
- [ ] X 버튼으로 모달 닫힘

### 2.2 설정 저장
- [ ] Spreadsheet ID 입력 후 저장 버튼 클릭
- [ ] Client ID 입력 후 저장 버튼 클릭
- [ ] 저장 성공 시 토스트 메시지 표시됨
- [ ] 페이지 새로고침 후 설정 유지됨

### 2.3 Google OAuth 로그인
- [ ] Client ID 설정 후 "Google 계정으로 로그인" 버튼 표시됨
- [ ] 로그인 버튼 클릭 시 Google 인증 팝업 열림
- [ ] 인증 성공 시 사용자 이메일 표시됨
- [ ] 인증 성공 시 동기화 버튼 활성화됨
- [ ] 로그아웃 버튼 클릭 시 세션 종료됨

---

## Phase 3: 타겟 등록 기능 테스트

### 3.1 타겟등록 탭 UI
- [ ] 필터 영역 표시됨 (월, 행선지, 모델 필터)
- [ ] 검색 입력 필드 표시됨
- [ ] "선택 항목 등록" 버튼 표시됨
- [ ] 오더 목록 테이블 표시됨

### 3.2 오더 선택
- [ ] 체크박스로 개별 오더 선택 가능
- [ ] 전체 선택 체크박스 동작
- [ ] 선택된 오더 수 표시됨
- [ ] 필터 적용 시 테이블 업데이트됨

### 3.3 타겟 등록
- [ ] 1개 이상 오더 선택 후 "등록" 버튼 클릭
- [ ] 등록 확인 다이얼로그 표시됨
- [ ] 등록 성공 시 토스트 메시지 표시됨
- [ ] 등록된 타겟이 모니터링현황 탭에 표시됨
- [ ] Google Sheets에 데이터 저장됨 (QMS_Targets 시트)

## Phase 4: 모니터링 현황 테스트

### 4.1 모니터링현황 탭 UI
- [ ] 등록된 타겟 목록 테이블 표시됨
- [ ] 각 타겟의 생산 진행률 표시됨
- [ ] 상태 배지 표시됨 (active/completed/cancelled)
- [ ] 우선순위 표시됨 (high/medium/low)

### 4.2 타겟 관리
- [ ] 타겟 상태 변경 가능 (completed, cancelled)
- [ ] 타겟 삭제 버튼 동작
- [ ] 삭제 확인 다이얼로그 표시됨
- [ ] 삭제 후 목록에서 제거됨

### 4.3 데이터 새로고침
- [ ] 새로고침 버튼 클릭 시 최신 데이터 로드됨
- [ ] 로딩 인디케이터 표시됨
- [ ] Google Sheets 데이터와 동기화됨

## Phase 5: 작업 지시 테스트

### 5.1 작업지시 탭 UI
- [ ] 타겟 선택 드롭다운 표시됨
- [ ] 작업 유형 선택 (검수/수정·재작업/커스텀)
- [ ] 작업 생성 폼 표시됨

### 5.2 작업 생성
- [ ] 타겟 선택 후 작업 유형 선택
- [ ] 작업 제목, 설명 입력
- [ ] 담당자 지정 (이메일)
- [ ] 마감일 설정
- [ ] 우선순위 설정 (high/medium/low)
- [ ] "작업 생성" 버튼 클릭
- [ ] 생성 성공 시 토스트 메시지 표시됨
- [ ] Google Sheets에 데이터 저장됨 (QMS_Tasks 시트)

### 5.3 작업 유형별 폼
- [ ] 검수(inspection): 기본 폼 표시됨
- [ ] 수정·재작업(rework): 추가 필드 표시됨
- [ ] 커스텀(custom): 커스텀 필드 입력 가능

## Phase 6: 작업 목록 테스트

### 6.1 작업목록 탭 UI
- [ ] 전체 작업 목록 테이블 표시됨
- [ ] 상태별 필터 동작 (pending/in_progress/completed/cancelled)
- [ ] 담당자별 필터 동작
- [ ] 타겟별 필터 동작

### 6.2 작업 상태 관리
- [ ] 작업 상태 변경 가능 (pending → in_progress → completed)
- [ ] 상태 변경 시 Google Sheets 업데이트됨
- [ ] 상태별 배지 색상 구분됨

### 6.3 작업 상세 보기
- [ ] 작업 행 클릭 시 상세 정보 표시됨
- [ ] 연관 타겟 정보 표시됨
- [ ] 작업 히스토리 표시됨

## Phase 7: 결과 관리 테스트

### 7.1 결과관리 탭 UI
- [ ] 완료된 작업 목록 표시됨
- [ ] 결과 입력 버튼 표시됨
- [ ] 기존 결과 목록 표시됨

### 7.2 결과 입력
- [ ] 결과 입력 폼 표시됨
- [ ] 결과 상태 선택 (pass/fail/partial)
- [ ] 합격/불합격 수량 입력

- [ ] 조치 내용 입력
- [ ] 사진 URL 입력 (선택)
- [ ] "결과 저장" 버튼 클릭
- [ ] 저장 성공 시 토스트 메시지 표시됨
- [ ] Google Sheets에 데이터 저장됨 (QMS_Results 시트)

### 7.3 결과 조회
- [ ] 저장된 결과 목록 표시됨
- [ ] 타겟별 결과 필터링 동작
- [ ] 결과 상세 보기 동작

## Phase 8: 실시간 동기화 테스트

### 8.1 자동 동기화 설정
- [ ] 설정 모달에서 자동 동기화 토글 ON
- [ ] 동기화 간격 선택 (15초/30초/1분/2분/5분)
- [ ] 설정 저장 후 자동 동기화 시작됨
- [ ] 동기화 간격에 따라 데이터 갱신됨

### 8.2 동기화 상태 표시
- [ ] 마지막 동기화 시간 표시됨
- [ ] 동기화 중 로딩 인디케이터 표시됨
- [ ] 동기화 오류 시 에러 메시지 표시됨

### 8.3 수동 동기화
- [ ] 동기화 버튼 클릭 시 즉시 동기화 실행
- [ ] 동기화 완료 후 데이터 갱신됨

- [ ] 타겟 100건 이상 시 정상 동작
